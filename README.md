# Banking Management System (Java)

ระบบบริหารจัดการบัญชีธนาคารเบื้องต้น พัฒนาด้วยภาษา Java โดยเน้นการประยุกต์ใช้หลักการเขียนโปรแกรมเชิงวัตถุ (Object-Oriented Programming - OOP) ในการจัดการข้อมูลผู้ใช้งานและบัญชีธนาคาร

## โครงสร้างและส่วนประกอบของระบบ
ระบบถูกออกแบบโดยแบ่งหน้าที่การทำงานผ่านคลาสต่างๆ ดังนี้:
- **User:** จัดการข้อมูลส่วนบุคคลของผู้ใช้งานระบบ
- **Account:** จัดการข้อมูลบัญชีเงินฝาก ยอดเงินคงเหลือ และการคำนวณพื้นฐาน
- **BankSystem:** ระบบควบคุมการทำงานหลัก (Core Logic) เช่น การฝากเงิน การถอนเงิน และการตรวจสอบสิทธิ์
- **Main:** ส่วนควบคุมการเริ่มต้นทำงานของโปรแกรม (Entry Point)

## คุณสมบัติของโปรแกรม
- ระบบจัดการข้อมูลผู้ใช้งานและเลขที่บัญชี
- ฟังก์ชันพื้นฐานทางการเงิน: ฝากเงิน (Deposit) และถอนเงิน (Withdraw)
- ระบบตรวจสอบยอดเงินคงเหลือในบัญชี
- การจัดเก็บและบริหารจัดการข้อมูลในรูปแบบ Object-oriented

## เทคโนโลยีที่ใช้
- **Language:** Java
- **Development Tool:** IntelliJ IDEA (รองรับไฟล์โครงการ .iml)
- **Programming Paradigm:** Object-Oriented Programming (OOP)

## ขั้นตอนการติดตั้งและรันโปรแกรม
1. คัดลอก (Clone) Repository นี้ลงในเครื่องของคุณ
2. เปิดโปรเจกต์ด้วย IntelliJ IDEA หรือ Java IDE ที่คุณใช้งาน
3. ตรวจสอบการตั้งค่า JDK (แนะนำเวอร์ชั่น 11 หรือสูงกว่า)
4. รันไฟล์ `Main.java` เพื่อเริ่มต้นการทำงานของระบบ
